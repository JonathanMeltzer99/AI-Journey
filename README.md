# AI Journey

This repository documents experiments, projects, and learning progress in artificial intelligence and machine learning.

The goal of this project is to explore AI development through hands-on experimentation with models, tools, and coding workflows.

---

## Project Purpose

AI Journey is a sandbox repository for:

- experimenting with AI tools and environments
- building small AI-related projects
- testing development workflows with AI coding agents
- documenting learning progress in AI engineering

The repository will gradually grow with experiments involving:

- machine learning
- large language models
- AI agents
- automation tools
